# Retail and Sales Performance Analysis

## 📊 Overview
This project analyzes retail customer data to uncover insights on churn, sales trends, and performance metrics using MySQL, and Power BI.

## 🔹 Features
- SQL views for churn and customer segmentation
- Power BI dashboards with KPIs and interactive visuals

## 🛠️ Tech Stack
- MySQL
- Power BI

## 🚀 How to Run
1. Clone the repository.
2. Import datasets into MySQL.
3. Connect Power BI to MySQL views.
4. Explore dashboards for insights.

## 📈 Key Visuals
- Churn by city
- Customer segments by spending
- Subscription type vs churn
- Total revenue and KPIs

## 📂 Dataset
E-commerce customer churn dataset (Kaggle).
